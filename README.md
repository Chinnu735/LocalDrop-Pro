# LocalDrop Pro

Ultra-secure, cross-platform local file drop Progressive Web App (PWA) using WebRTC and Framer Motion. 
Built to serve as an "AirDrop for everything" (Windows, macOS, Linux, Android, iOS) with a gorgeous Void Black and Neon Purple aesthetic.

## Features
- **True Cross-Platform:** Runs in any browser via PWA.
- **Peer-to-Peer:** Files never touch a central server. Transferred directly via WebRTC.
- **E2E Encrypted:** Built-in DTLS/SRTP WebRTC encryption with PIN pairing.
- **Pro UI:** Physics-based Framer Motion animations.

## Tech Stack
- **Client:** React, TypeScript, Vite, TailwindCSS, Framer Motion
- **Server:** Node.js, Express, Socket.io (Signaling only)
